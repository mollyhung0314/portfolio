項目介紹 / Project Introduction

一款為打工族所設計的APP，求職者可藉由地圖搜尋工作，透過聊天室與雇主即時聯絡，讓徵才與求職不再受限於時間、地點。

色彩搭配 / Color Match

主色選用代表熱情、鬥志的紅色，搭配黑色，襯托簡約、時尚質感

#E2231B
#2a2a2a

UI 視覺介面設計 / UI Design

主頁面

  • 可透過切換紐，將職缺切換為卡片式或列表式

  • 列表式職缺分類可水平滑動

  • 愛心按鈕依使用者操作習慣放置於職缺右上角， 並加入自然陰影效果，刺激點擊慾望

推薦職缺

  • 利用卡片式設計呈現，讓使用者可以在卡片堆中找尋自己感興趣的職缺

  • 精簡化職缺內容，讓使用者能輕易、快速的理解資訊
